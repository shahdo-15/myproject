# 🚀 Laravel Authentication System

## 📌 Overview
This project is a simple and secure authentication system built using Laravel.  
It allows users to register, login, and access protected pages.

---

## ✨ Features
✅ User Registration (Name, Email, Address)  
✅ Secure Password Hashing  
✅ Login System  
✅ Middleware Protection (auth & guest)  
✅ Welcome Page with User Data  
✅ Logout Functionality  

---

## 🛠️ Technologies Used
- PHP
- Laravel
- MySQL
- Blade

---

## ▶️ How to Run the Project

- composer install
- php artisan serve
